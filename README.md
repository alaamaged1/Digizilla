# Digizilla - Odoo v17 Addon

**Digizilla** is a custom Odoo module to manage records with notes, comments, and PDF reporting.

## Features

- Model: `digizilla.record`
- Views: Kanban, Form, List
- Fields:
  - Name* (mandatory)
  - Gender
  - Country
  - Joining Date
  - Tags
  - Customers (multi-select from default Odoo Customers)
  - Company* (single-select from default Odoo Companies)
  - Notes (long text, separate tab)
  - Comments (short text, separate tab)
- Tracks changes in all fields (Messages & Logger)
- Hides the create button in the form view
- Generates PDF reports of records
